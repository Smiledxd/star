# Inca Kola Concept - Landing Page

Este proyecto es una **propuesta conceptual y académica** para el rediseño de una landing page inspirada en la marca Inca Kola. El objetivo principal fue poner en práctica técnicas avanzadas de maquetación web y diseño visual.

## 🚀 Objetivo del Proyecto

El desarrollo se centró en recrear la identidad visual de una marca icónica, aplicando conceptos modernos de experiencia de usuario (UX) y optimización de código.

### Habilidades Técnicas Aplicadas:

- **Estructura Semántica:** Uso de HTML5 para una base sólida y accesible.
- **Diseño Responsivo:** Implementación de layouts flexibles mediante **Flexbox**.
- **Estética Visual:** Manejo de paletas de colores corporativos y tipografías personalizadas.
- **Animaciones:** Efectos sutiles en CSS para mejorar la interacción con el usuario.

## 🛠️ Tecnologías Utilizadas

- **HTML5 & CSS3**: Estructura y diseño responsivo.
- **[Servor](https://github.com/lukejacksonn/servor)**: Servidor local de desarrollo rápido.
- **[Oxlint](https://oxc.rs/docs/guide/usage/linter.html)**: Calidad del código.
- **[PNPM](https://pnpm.io/)**: Gestor de paquetes eficiente.
